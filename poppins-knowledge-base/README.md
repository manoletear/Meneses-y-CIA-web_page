# Poppins Knowledge Base

Repositorio central de conocimiento para el desarrollo de **Poppins**, plataforma de gestión de contratos laborales y cumplimiento previsional en Chile.

## 📋 Propósito

Este repositorio contiene toda la documentación normativa, código de referencia y recursos de entrenamiento necesarios para que el equipo "anigravity" desarrolle Poppins con los más altos estándares de cumplimiento legal y técnico.

## 🗂️ Estructura

- /docs - Documentación normativa, resúmenes de leyes, guías operativas
- /src - Código fuente (validadores, parsers, generadores, workers)
- /training - Recursos para entrenar Google Antigravity (prompts, workflows)
- /research - Enlaces oficiales y fuentes primarias
- CHANGELOG-NORMATIVO.md - Registro de cambios en la normativa
- ENLACES-COMPLETOS.md - Listado maestro de enlaces oficiales

## 🚀 Primeros Pasos

1. Revisa el GLOSARIO-CRITICO.md en /docs
2. Familiarízate con los enlaces oficiales en ENLACES-COMPLETOS.md
3. Explora el código en /src para entender las implementaciones base
4. Usa los prompts en /training/prompts para interactuar con Antigravity

## 📌 Mantenimiento

Este repositorio debe actualizarse mensualmente siguiendo el proceso descrito en CHANGELOG-NORMATIVO.md.
